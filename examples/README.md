# Examples

This folder contains a sample architecture diagram for a Beckhoff C6920 + TwinCAT 3.1 control system.

Files:

- `beckhoff_c6920_twincat_architecture.drawio` - editable draw.io source
- `beckhoff_c6920_twincat_architecture.svg` - SVG generated from the draw.io file
- `beckhoff_c6920_twincat_architecture.png` - rendered preview

Regenerate the SVG:

```bash
python ../scripts/drawio_to_svg.py beckhoff_c6920_twincat_architecture.drawio beckhoff_c6920_twincat_architecture.svg
```
